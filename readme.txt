Concert-alert has to purpose to get the music what you lesson and send a periodic email with the concerts present in you city.

Create an apps https://developers.deezer.com/api
Ask a Songkick api http://www.songkick.com/developer

Fill the config file config.js

Log with your deezer account in http://localhost:3000

The data is displayed in http://localhost:3000/history
